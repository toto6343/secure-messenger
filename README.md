# 🔒 Secure Messenger

안전하고 프라이버시를 존중하는 End-to-End 암호화 메신저

## ✨ 주요 기능

### 보안
- **End-to-End 암호화**: 모든 메시지는 클라이언트 측에서 암호화되어 서버도 내용을 볼 수 없음
- **고급 암호화 알고리즘**: RSA-OAEP 2048 및 AES-GCM 256 기반 강력한 보안
- **대화별 암호화**: 각 대화마다 고유한 암호화 키 사용
- **메시지 자동 삭제 (Burn-after-reading)**: 읽은 후 일정 시간(5초~1시간)이 지나면 자동 삭제되는 타이머 기능
- **암호화 백업 및 클라우드 연동**: 대화 키와 데이터를 사용자의 비밀번호로 암호화하여 클라우드(시뮬레이션) 또는 파일로 안전하게 백업

### 기능
- **실시간 메시징**: Socket.io 기반 실시간 채팅
- **친구 추가 및 관리**: 사용자 검색을 통해 친구를 추가하고, 전용 목록에서 관리 및 빠른 채팅 시작 가능
- **비밀번호 찾기 및 재설정**: 이메일 인증 토큰을 통한 안전한 비밀번호 복구 프로세스
- **파일 전송**: 이미지 및 문서 파일을 암호화하여 안전하게 전송
- **메시지 검색**: 암호화된 메시지를 로컬에서 복호화하여 실시간 검색 지원
- **음성/영상 통화**: WebRTC 기반 P2P 통화 지원
- **그룹 채팅 관리**: 관리자 권한 설정, 멤버 추가/제외 기능
- **메시지 읽음 상세 확인**: 누가 언제 메시지를 읽었는지 상세 리스트 확인 가능
- **온라인 상태 및 입력 중 표시**: 실시간 사용자 상태 트래킹

### 플랫폼
- **모바일**: React Native (iOS & Android) - 친구 관리 및 비밀번호 찾기 포함 고도화 완료
- **데스크톱**: Electron (Windows, macOS, Linux) - 탭 기반 사이드바 및 친구 관리 포함 고도화 완료
- **크로스 플랫폼**: 동일한 계정으로 모든 기기에서 사용 가능

## 🏗️ 프로젝트 구조

```
secure-messenger/
├── backend/                 # Node.js 백엔드 서버 (친구 관리 및 비밀번호 재설정 API 포함)
│   ├── server.js           # 메인 서버 파일 (이메일 정규화 및 보안 강화 적용)
│   └── uploads/            # 암호화된 파일 저장소
│
├── mobile/                 # React Native 모바일 앱 (친구 화면 및 비밀번호 찾기 구현)
│   ├── screens/            # Friends, ForgotPassword, Chat 등
│   └── utils/              # 강력한 E2E 암호화 및 백업 엔진
│
└── desktop/                # Electron 데스크톱 앱
    └── renderer/           # 데스크톱 전용 고도화 UI (친구/대화 탭 및 비밀번호 찾기 모달)
```

## 🔐 보안 아키텍처

### 암호화 백업 로직
1. 사용자가 백업 비밀번호를 설정합니다.
2. 시스템은 로컬의 모든 `대화 키`와 `세션 데이터`를 수집합니다.
3. 설정한 비밀번호로 AES-GCM 256 암호화를 수행하여 2차 암호화된 백업 파일을 생성합니다.
4. 이 데이터는 클라우드 저장소(AWS S3 등)에 동기화되거나 로컬 파일로 저장됩니다.
5. 복구 시에는 해당 비밀번호가 있어야만 데이터를 해독할 수 있어, 서버 관리자도 백업 내용을 볼 수 없습니다.
